# 欢迎来到我的博客

你好！这是我的第一篇博客文章。

## 关于这个网站

这个个人主页是我用纯 HTML、CSS 和 JavaScript 搭建的。它支持：

- **自动深色/浅色模式**：跟随系统设置自动切换
- **Markdown 博客系统**：文章以 `.md` 文件形式存放，通过 `marked.js` 渲染
- **响应式设计**：在手机、平板、桌面端都能获得良好的阅读体验

## 我会在这里写什么

这个博客主要记录：

1. **学习笔记** — 课程内容的整理和复习
2. **项目总结** — 做过的项目的经验与反思
3. **读书心得** — 阅读技术书籍的摘录与思考
4. **技术探索** — 尝试新工具、新框架的记录

## 搭建过程

搭建这个网站的过程本身也是一次很好的学习。从设计布局、选择配色，到实现 Markdown 渲染和响应式适配，每一步都让我对前端开发有了更深的理解。

> 知识在分享中增长，希望这里的内容对你也有帮助。
