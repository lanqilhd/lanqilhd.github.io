# 机器学习入门笔记

整理了一些机器学习基础概念和常用算法的入门笔记。

## 什么是机器学习

机器学习是人工智能的一个分支，它使计算机能够在没有明确编程的情况下从数据中学习。Arthur Samuel 将其定义为"一个让计算机无需明确编程即可学习的研究领域"。

## 主要分类

### 监督学习 (Supervised Learning)

在监督学习中，我们有一个带有标签的数据集，模型学习从输入到输出的映射关系。

常见算法：
- **线性回归** — 预测连续值
- **逻辑回归** — 二分类问题
- **决策树** — 树形结构的分类与回归
- **支持向量机 (SVM)** — 寻找最优分类超平面
- **神经网络** — 通过多层非线性变换学习复杂模式

### 无监督学习 (Unsupervised Learning)

在无监督学习中，数据没有标签，模型需要自行发现数据中的结构和模式。

常见算法：
- **K-Means 聚类** — 将数据分为 K 个簇
- **PCA (主成分分析)** — 降维
- **自编码器 (Autoencoder)** — 通过重构学习数据表示

### 强化学习 (Reinforcement Learning)

智能体通过与环境的交互来学习最优策略，通过奖励信号来指导行为。

## 关键概念

1. **过拟合与欠拟合**
   - 过拟合：模型在训练集上表现很好，但在测试集上很差
   - 欠拟合：模型在训练集和测试集上都表现不佳

2. **偏差-方差权衡**
   - 高偏差 → 欠拟合
   - 高方差 → 过拟合

3. **交叉验证**
   - 将数据分为 K 折，轮流用 K-1 折训练、1 折验证
   - 帮助评估模型的泛化能力

## 推荐学习路径

1. 先掌握数学基础：线性代数、概率论、微积分
2. 学习 Python 和 NumPy、Pandas
3. 从 Scikit-learn 开始实践经典算法
4. 逐步深入深度学习框架 (PyTorch / TensorFlow)
